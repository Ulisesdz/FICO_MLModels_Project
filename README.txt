PARA VER LA EJECUCIÓN COMPLETA DEL PROYECTO CON LOS RESULTADOS OBTENIDOS EN EL INFORME, ABRIR
EL ARCHIVO:
- global.ipynb

EN CASO DE QUERER EJECUTAR EL PROYECTO DE MANERA MODULAR, EJECUTAR EL ARCHIVO:
- main.py

ESTE ARCHIVO ABRIRÁ UN MENU QUE PERMITE EJECUTAR LOS ARCHIVOS DE LAS DIFERENTES PARTES DEL
PROYECTO

- PARTE 1: ANALISIS Y DIVISIÓN DE LOS DATOS: data_analysis.py
- PARTE 2: MODELOS DE CLASIFICACION: classification.py
	   MODELOS DE CLASIFICACION CON SKLEARN: classification_sklearn.py
	   COMPARATIVA DE MODELOS: models_comparison.py
- PARTE 3: MODELOS NO SUPERVISADOS: unsupervised.py

LA PARTE 2 SE EJECUTA DE MANERA PREDEFINIDA CON EL SET DE DATOS USADO POR EL ALUMNO, LOS CUALES
SE GUARDARON EN LA CARPETA DATASETS PARA QUE EL PROFESORADO PUEDA OBTENER LOS MISMOS REULTADOS QUE
LOS COMENTADOS EN EL INFORME. PARA CAMBIAR ESTO, HABRÁ QUE VARIAR LAS RUTAS DE LOS ARCHIVOS QUE
SE DECLARAN AL PRINCIPIO DE CADA ARCHIVO PYTHON.
